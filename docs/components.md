# Components

## Layout

**NavBar** - Top navigation bar with links to main pages

**Footer** - Bottom footer with contact info and social links

**Container** - Max-width wrapper for page content

**BackgroundOverlay** - Gradient background overlay effect

**SmoothScroll** - Lenis smooth scrolling wrapper

**PageLoadOverlay** - Loading screen overlay

## Home Page

**HeroSection** - Landing section with main heading and CTA

**WhatWeOffer** - Section showing what the club offers

**ImageCarousel** - Auto-scrolling image carousel

**HomeImageCaptions** - Captions for carousel images

**ActivityGroups** - Activity groups display (legacy, see AboutChunk)

**Benefits** - Member benefits section

**BigThreeBlurb** - Highlights three main areas

**Sponsorship** - Sponsor logos and info

## About Page

**AboutLanding** - About page hero with officers group photo

**AboutMissionText** - Mission statement text block

**AboutMissionSlider** - Image slider for mission section

**AboutMissionImageText** - Image + text section

**AboutChunk** - Activity groups, committees, and certifications sections

**Leadership** - Officers and activity leaders grid

**AboutBlurb** - Legacy about text component

**AboutMission** - Legacy about mission component

## Join Page

**JoinLanding** - Join page hero section

**JoinGuide** - Step-by-step guide to joining

## Partner Page

**PartnerLanding** - Partner page hero

**PartnerGuide** - Information for potential partners

## Dashboard

**Dashboard** - Main dashboard container

**DashboardSidebar** - Side navigation for dashboard

**Login** - Login form component

## Utility

**Accordion** - Expandable/collapsible content

**Calendar** - Event calendar display

**FeatureSlider** - Feature showcase slider

**TitleContainer** - Styled title wrapper

**CenteredMessage** - Centered message display

**LoadingPage** - Full page loading state

**Logo** - Club logo component

**NavSpacer** - Spacing element for fixed nav

**ObfuscatedLink** - Anti-scraper email/link obfuscation

**Report** - Report display component

**TermPicker** - Semester term selector

**TooltipText** - Tooltip wrapper

**Watermark** - Page watermark element

**Anchor** - Scroll anchor link
